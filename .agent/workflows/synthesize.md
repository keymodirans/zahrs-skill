# /synthesize

Panggil MCP tool `index_research_synthesizer` untuk combine research.

## Usage
```
/synthesize <research files comma-separated>
```

## Steps
1. Trigger MCP tool `mcp_zahrs-skill_index_research_synthesizer`
2. Combine outputs dari multiple researchers
3. Output unified SUMMARY.md

## Example
```
/synthesize ".planning/research/STACK.md,.planning/research/FEATURES.md"
```
