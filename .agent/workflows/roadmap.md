# /roadmap

Panggil MCP tool `create_roadmap` untuk break project into phases.

## Usage
```
/roadmap <project name>
```

## Steps
1. Trigger MCP tool `mcp_zahrs-skill_create_roadmap`
2. Analyze requirements, create phase breakdown
3. Output ROADMAP.md dengan milestones

## Example
```
/roadmap my-saas-app
/roadmap mobile-banking
```
