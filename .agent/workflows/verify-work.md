# /verify-work

Panggil MCP tool `verify_work` untuk check work deliverables.

## Usage
```
/verify-work <work description>
```

## Steps
1. Trigger MCP tool `mcp_zahrs-skill_verify_work`
2. Goal-backward analysis
3. Report VERIFIED atau gaps

## Example
```
/verify-work "Implemented login dengan OAuth"
```
