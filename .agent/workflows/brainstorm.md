# /brainstorm

Panggil MCP tool `brainstorm` untuk sesi ideation interaktif.

## Usage
```
/brainstorm <topic>
```

## Steps
1. Trigger MCP tool `mcp_zahrs-skill_brainstorm`
2. Agent akan memimpin diskusi interaktif
3. Explore pros/cons dan alternatif
4. Synthesize hasil diskusi

## Example
```
/brainstorm Microservices vs Monolith untuk aplikasi ini
/brainstorm Fitur gamifikasi untuk user retention
```
