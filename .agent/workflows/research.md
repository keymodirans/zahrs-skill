# /research

Panggil MCP tool `research_project` untuk deep research.

## Usage
```
/research <project description>
```

## Steps
1. Trigger MCP tool `mcp_zahrs-skill_research_project`
2. Research stack, features, architecture, pitfalls
3. Output research files ke .planning/research/

## Example
```
/research E-commerce platform with payment integration
/research Real-time chat application with WebSocket
```
