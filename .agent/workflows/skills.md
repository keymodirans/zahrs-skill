# /skills

Panggil MCP tool `list_skills` untuk tampilkan semua skills.

## Usage
```
/skills [folder]
```

## Steps
1. Trigger MCP tool `mcp_zahrs-skill_list_skills`
2. Display table of skills dengan name, file, description

## Example
```
/skills
/skills references
/skills templates
```
