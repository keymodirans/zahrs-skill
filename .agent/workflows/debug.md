# /debug

Panggil MCP tool `debug` untuk investigate bugs.

## Usage
```
/debug <bug description>
```

## Steps
1. Trigger MCP tool `mcp_zahrs-skill_debug` dengan bug_description dari user
2. Follow workflow dari skill content yang di-return
3. Report findings ke user

## Example
```
/debug Login gagal redirect ke dashboard, stuck di login page
```
