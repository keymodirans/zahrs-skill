# /phase-research

Panggil MCP tool `index_phase_researcher` untuk phase-specific research.

## Usage
```
/phase-research <phase name> [context]
```

## Steps
1. Trigger MCP tool `mcp_zahrs-skill_index_phase_researcher`
2. Research implementation approach
3. Output RESEARCH.md

## Example
```
/phase-research authentication-system
/phase-research "payment integration" "Using Stripe API"
```
